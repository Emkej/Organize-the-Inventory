# AGENTS.md

## Build Requirement
- After any code change, run a game build with the repo build script before finishing the task.
- If the build was not run or failed, say so explicitly and do not present the change as complete.

## Commit Message Convention
- Use Conventional Commit format.
- Use imperative mood with an uppercase first letter in the subject after the type prefix.
- Example: `feat: Add selected-member delete-all action`
